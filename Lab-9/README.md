# Function & Pointer Based Scenario Programs

---

## 🎯 Objective
The objective of this lab is to practice and understand functions and pointers in C programming through real-life scenario-based problems.

This lab focuses on:
- Writing modular programs using functions
- Passing arrays and values to functions
- Using pointers and pointer arithmetic
- Working with 2D arrays using pointers
- Understanding void pointers for generic data handling
- Using function pointers for runtime decision making
