## 📌 Tasks Overview

### Q1 — Tower Countdown (Recursion)
- Compute total power consumption of a building
- Each floor doubles the previous one
- Uses recursive function

---

### Q2 — Password Strength Checker (Recursion)
- Count uppercase letters in a string
- Count digits in a string
- Uses recursive character traversal

---

### Q3 — Maze Escape (Recursion + Paths)
- Find number of ways to reach position N
- Moves allowed: 1 step or 2 steps
- Print all possible paths

---

### Q4 — Hospital Patient Registry (Structures)
- Store patient records using struct
- Display all records
- Search patient by ID

---

### Q5 — University Course Catalog (Nested Structures)
- Department inside Course struct
- Store multiple courses
- Filter courses by department code

---

### Q6 — E-Commerce Order Tracker (Structures + Functions)
- Store order details
- Compute total bill (quantity × unit price)
- Filter orders by status (Pending, Shipped, Delivered)
