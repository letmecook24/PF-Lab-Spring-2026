# C Programming Mini Projects – Loop-Based Tasks

This folder contains 12 C programming tasks focusing on loops (`for`, `while`, `do-while`) and control structures.  
These programs help practice **efficient iteration, conditional execution, and dynamic program control**.

---

## Objectives

- **Iterate Over Data Structures:** Efficiently traverse arrays, linked lists, or other collections without repetitive code.  
- **Terminate on Specific Conditions:** Execute loops only while certain conditions are met, stopping once the desired outcome occurs.  
- **Reduce Code Complexity:** Simplify operations and minimize code lines for cleaner, more logical repetition.  
- **Support Event-Driven Programming:** Continuously monitor inputs or triggers in real-time applications without terminating.  
- **Flexible Program Execution:** Control loop flow dynamically using conditions, including `break` and `continue` statements.

## Task Descriptions

1. **Gym Membership Check-In System** – Track check-ins for 25 gym members using a `for` loop.  
2. **Mobile Data Usage Tracker** – Deduct data usage continuously until balance is exhausted (`while` loop).  
3. **Door Access Security System** – Unlock door only when correct code is entered (`do-while` loop).  
4. **Total Distance Calculator** – Calculate total distance covered over N days (`for` loop).  
5. **Electricity Bill Monitoring System** – Compute average daily units consumed until sentinel value (`while` loop).  
6. **Salary Increment Table Generator** – Print salary for next 10 years with 5% increment (`for` loop).  
7. **Library Management Mini System** – Menu-driven system for issuing/returning books (`do-while` loop).  
8. **PIN Verification Game** – Verify user PIN until it matches the correct one (`while` loop).  
9. **Event Ticket Pricing System** – Calculate total revenue from N VIP tickets with incremental pricing (`for` loop).  
10. **Employee Performance Classification System** – Categorize employees by rating until sentinel value (`while` loop).  
11. **Restaurant Billing System** – Add food items continuously and apply discount if applicable (`do-while` loop).  
12. **Parking Lot Space Monitoring System** – Track car entries until full or operator stops (`while` loop).
