# Lab 07: Introduction to One-Dimensional Arrays with Loops

---

## 📌 Lab Overview
This lab focuses on understanding **one-dimensional arrays in C** and how they work with **loops** to process multiple data elements efficiently.  
The exercises demonstrate how arrays store data in contiguous memory and how loops allow us to access, modify, and analyze that data.

---

## 🎯 Objectives
- Understand the concept of **one-dimensional arrays in C**
- Learn how arrays are **stored in memory and accessed using indexes**
- Use **loops to iterate through arrays efficiently**
- Work with **character arrays (strings)**
- Learn how to **modify elements inside arrays**
- Understand **scanset and negated scanset in scanf()**
- Practice solving problems using **arrays and loops**

---

## 🧠 Concepts Covered
- One-Dimensional Arrays
- Looping through arrays using `for` loops
- Searching elements in arrays
- Finding **minimum and maximum values**
- Working with **character arrays**
- Case conversion (Uppercase ↔ Lowercase)
- Handling duplicate elements in arrays
- Using **scanf scansets `%[]`**
- Using **negated scansets `%[^...]`**

---

## 📚 What We Learned
- Arrays allow us to **store multiple values of the same type** in a single variable
- Loops help **automate repetitive tasks** when working with arrays
- Array elements are accessed using **indexes starting from 0**
- Character arrays can be used to **store and manipulate strings**
- Programs can **search, modify, and analyze data** stored in arrays
- The **scanset feature of scanf()** allows selective input handling
- Using arrays and loops makes programs **more efficient and readable**
